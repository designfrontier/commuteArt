commuteArt
==========

Commute Art: A generative art project

Commute Art is an attempt to turn the acclerometers available in devices with web access into a generative art project. That uses the bumps and movements of your daily commute to create pieces of art.

Technologies
===========
This is a ready to deploy app that will run on heroku. The actual project is running at http://commuteart.herokuapp.com for now.
